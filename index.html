WiFi QR Wizard for Android Enterprise Enrollment with Intune

Adds WiFi Support to your Enrollment-QR-Code













To view and export QR codes from Microsoft Intune, please follow these steps:

  1. Log in to the Microsoft Endpoint Manager admin center.
  2. Navigate to Devices > Enroll devices > Corporate device identifiers.
  3. Find the section for QR code tokens and select the token you wish to export.
  4. Click on the 'Export' button to download the QR code's JSON file.

Use this JSON file to generate a QR code on this page for device configuration.
